⚠️  THIS IS THE MOST IMPORTANT GUARDRAIL IN THIS DOCUMENT:

  The discovery engine MUST be a standalone Python script.
  NO Claude API calls. NO LLM calls. NO AI agents.
  NO web searches through Claude.
  JUST direct HTTP requests to public government data APIs.

  This script runs on the server with: python discovery_engine.py
  It costs ZERO Claude credits to run.

Create: discovery_engine.py

ARCHITECTURE:
  The script checks 3 tiers of data sources, in order.
  Each tier eliminates cities from later tiers.

  TIER 1: STATE-LEVEL (check first — one hit = hundreds of cities)
  ----------------------------------------------------------------
  For each of the 50 US states:
    - Check Socrata: search api.us.socrata.com for statewide
      building permit datasets
    - Check ArcGIS Hub: search for state-level permit services
    - Check data.gov (CKAN): search for state permit datasets

  If a state source is found and has recent data:
    → Extract the list of cities/counties covered
    → Mark all those cities as "covered by state source"
    → Log the source details

  TIER 2: COUNTY-LEVEL (check next — one hit = dozens of cities)
  ----------------------------------------------------------------
  For the top 200 US counties by population:
    Skip counties where ALL cities are already covered by Tier 1.

    - Check Socrata: data.{county}.gov permit datasets
    - Check ArcGIS: county GIS portals
    - Check data.gov

  If a county source is found:
    → Extract covered cities
    → Mark as "covered by county source"
    → Log the source

  TIER 3: CITY-LEVEL (check remaining cities by population)
  ----------------------------------------------------------
  For each city in us_cities, ordered by population descending:
    Skip if already covered by Tier 1 or Tier 2.
    Skip if already in prod_cities.

    - Check Socrata:
        GET https://api.us.socrata.com/api/catalog/v1
          ?q={city_name}+building+permits
          &limit=5
        Filter results for datasets containing "permit" in name
        If found: test with GET {endpoint}?$limit=1&$order=date DESC

    - Check ArcGIS Hub:
        GET https://hub.arcgis.com/api/v3/search
          ?q={city_name}+{state}+building+permits
          &filter[type]=Feature Service
        If found: test the FeatureServer/0/query endpoint

    - Check data.gov (CKAN):
        GET https://catalog.data.gov/api/3/action/package_search
          ?q={city_name}+building+permits
          &rows=5
        If found: check for downloadable/API-accessible resources

  For each discovered source, log to discovery_results.csv:
    city, state, population, source_type, source_scope,
    endpoint_url, dataset_name, has_recent_data (bool),
    newest_record_date, field_names, record_count, checked_at

IMPORTANT IMPLEMENTATION DETAILS:

  1. RATE LIMITING:
     - 2 seconds between API calls minimum
     - If you get a 429 (rate limited), back off 30 seconds
     - If you get 3 consecutive 429s, back off 5 minutes

  2. RESUMABLE:
