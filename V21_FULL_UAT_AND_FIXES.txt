BUG #1 — JSON COORDINATES DISPLAYED AS ADDRESSES
------------------------------------------------------------------------
Severity: CRITICAL
Count: 1,814 permits affected
Location: Primarily Silver Spring, MD area permits

PROBLEM:
Some permits have raw JSON objects stored in the address field instead
of parsed street addresses. They display on the site like:
  {'latitude': '39.028879', 'longitude': '-77.103714', 'human_address': '...'}

This looks broken and unprofessional to any visitor.

ROOT CAUSE:
The Socrata collector receives location fields as JSON objects containing
lat/lon and a human_address sub-field. The collector stores the raw JSON
string instead of extracting the human_address value.

FIX (two parts):

Part A — Fix the collector to parse JSON addresses going forward:
  In the Socrata collector's normalize function, when extracting the
  address field, check if the value looks like a JSON object (starts
  with '{' or contains 'human_address'). If so, parse it:

    import json
    raw_address = record.get('address') or record.get('location') or ''
    if isinstance(raw_address, dict):
        raw_address = raw_address.get('human_address', '') or str(raw_address)
    elif isinstance(raw_address, str) and raw_address.startswith('{'):
        try:
            parsed = json.loads(raw_address.replace("'", '"'))
            raw_address = parsed.get('human_address', '') or raw_address
        except:
            pass

Part B — Clean existing bad data on prod DB:
  Run on Render Shell:

  python3 -c "
  import sqlite3, json, re
  conn = sqlite3.connect('/var/data/permitgrab.db')
  rows = conn.execute(\"SELECT rowid, address FROM permits WHERE address LIKE '%human_address%' OR address LIKE '%coordinates%'\").fetchall()
  fixed = 0
  for rowid, addr in rows:
      try:
          parsed = json.loads(addr.replace(chr(39), chr(34)))
          human = parsed.get('human_address', '')
          if human:
              conn.execute('UPDATE permits SET address=? WHERE rowid=?', (human, rowid))
              fixed += 1
      except: pass
  conn.commit()
  print('Fixed', fixed, 'of', len(rows), 'JSON addresses')
  "

VERIFY: Search for permits in Silver Spring, MD — addresses should now
show normal street addresses, not JSON blobs.
