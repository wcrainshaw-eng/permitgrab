EXAMPLE QUERIES
---------------

1. Get all 2026 permits (first 1000):
https://services.arcgis.com/8Pc9XBTAsYuxx9Ny/arcgis/rest/services/miamidade_permit_data/FeatureServer/0/query?where=PermitIssuedDate%20%3E%3D%20%272026-01-01%27&outFields=*&returnGeometry=true&f=json

2. Get recent permits with specific fields:
https://services.arcgis.com/8Pc9XBTAsYuxx9Ny/arcgis/rest/services/miamidade_permit_data/FeatureServer/0/query?where=PermitIssuedDate%20%3E%3D%20%272025-01-01%27&outFields=PermitNumber,PermitIssuedDate,PropertyAddress,City,PermitType,EstimatedValue&returnGeometry=false&orderByFields=PermitIssuedDate%20DESC&f=json

3. Search for Hialeah area permits (by address):
https://services.arcgis.com/8Pc9XBTAsYuxx9Ny/arcgis/rest/services/miamidade_permit_data/FeatureServer/0/query?where=PropertyAddress%20LIKE%20%27%25HIALEAH%25%27%20AND%20PermitIssuedDate%20%3E%3D%20%272026-01-01%27&outFields=*&returnGeometry=true&f=json

4. Get permits by type:
https://services.arcgis.com/8Pc9XBTAsYuxx9Ny/arcgis/rest/services/miamidade_permit_data/FeatureServer/0/query?where=PermitType%20%3D%20%27BLDG%27%20AND%20PermitIssuedDate%20%3E%3D%20%272026-01-01%27&outFields=*&f=json

5. Get high-value permits (over $1M):
https://services.arcgis.com/8Pc9XBTAsYuxx9Ny/arcgis/rest/services/miamidade_permit_data/FeatureServer/0/query?where=EstimatedValue%20%3E%201000000%20AND%20PermitIssuedDate%20%3E%3D%20%272026-01-01%27&outFields=*&f=json

QUERY PARAMETERS
----------------
- where: SQL-like WHERE clause for filtering
- outFields: Comma-separated field names or * for all
- returnGeometry: true/false to include spatial coordinates
- resultRecordCount: Limit results (max 1000 per request)
- orderByFields: Field name followed by ASC or DESC
- f: Output format (json, geojson, pjson)

PERMIT TYPE CODES
-----------------
- BLDG: Building permits
- ELEC: Electrical permits
- MBLD: Master building permits
- MECH: Mechanical permits
- PLUM: Plumbing permits
- ROOF: Roofing permits
