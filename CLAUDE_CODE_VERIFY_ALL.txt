PROJECT OVERVIEW
========================================================================

PermitGrab is a Flask web app deployed on Render.com (free tier, Docker) at permitgrab.com. It collects building permit data from city open data APIs (Socrata, ArcGIS) and presents them as contractor leads. The Stripe product is PermitGrab Professional at $149/month.

GitHub repo: wcrainshaw-eng/permitgrab (main branch, auto-deploys to Render)
Domain: permitgrab.com (Cloudflare DNS)
Stripe: Live keys in Render env vars
SendGrid: API key in Render env vars
