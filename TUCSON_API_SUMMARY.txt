- VALUE               : Project value (dollars)
- SQUAREFEET          : Square footage

LOCATION FIELDS:
- LAT                 : Latitude
- LON                 : Longitude
- WARD                : City ward

LINKS:
- CSS_URL             : Citizen portal link

- PRO_URL             : Pro portal link

METADATA:
- PREFIX              : Permit prefix (TC-RES, TC-COM, etc.)
- SOURCE              : Data source
- ACTIVE              : Active status ("Yes"/"No")
- StructureType       : Structure type description
- CensusCode          : Census code

MULTI-FAMILY ONLY:
- DwellingUnits       : Number of dwelling units
- DwellingBldgs       : Number of dwelling buildings

PYTHON QUICK START

import requests
from datetime import datetime

# Basic query
url = "https://mapdata.tucsonaz.gov/arcgis/rest/services/PublicMaps/PermitsCode/MapServer/85/query"
params = {
    'where': "APPLYDATE >= date'2026-01-01'",
    'outFields': '*',
    'f': 'json',
    'resultRecordCount': 10
}

response = requests.get(url, params=params)
data = response.json()

# Process results
for feature in data.get('features', []):
    attrs = feature['attributes']
    print(f"Permit: {attrs['NUMBER']}")
    print(f"Address: {attrs['ADDRESS']}")
    print(f"Status: {attrs['STATUS']}")

    # Convert timestamp
    apply_date = datetime.fromtimestamp(attrs['APPLYDATE'] / 1000)
    print(f"Applied: {apply_date.strftime('%Y-%m-%d')}")
    print()
